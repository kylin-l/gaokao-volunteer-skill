## 志愿模式速查

| 模式 | 典型省份 | 填报单位 | 关键规则 |
|------|----------|----------|----------|
| 院校专业组 | 新高考多数省 | 1 院校 + 1 专业组 + 组内若干专业 | 组内调剂；选科按组要求 |
| 院校+专业 | 部分新高考省 | 1 院校 + 1 专业 | 无组内调剂，专业级差仍可能存在 |
| 传统平行志愿 | 部分仍分科类省份 | 1 院校 + 若干专业 | 是否服从专业调剂决定退档风险 |

## 冲稳保位次换算

### 基本约定

1. **位次数字越小，排名越靠前**（第 5000 名优于第 15000 名）
2. **目标校录取最低位次**：该校/专业（组）近年录取的最靠后（数字最大）考生位次
3. 比较时以**位次**为主，分数仅作辅助

### 偏差率公式

```
位次偏差率 = (目标校录取最低位次 - 考生位次) / 考生位次
```

**解读**（考生位次 10000 为例）：

| 目标校最低位次 | 偏差率 | 含义 | 典型层级 |
|----------------|--------|------|----------|
| 8000 | -20% | 校更难，你位次偏后 | **冲** |
| 10000 | 0% | 与往年边界接近 | **稳** |
| 13000 | +30% | 校更易，你位次偏前 | **保/垫** |

- **偏差率 < 0**：目标校录取位次数字更小 → 校更难 → 偏**冲**
- **偏差率 > 0**：目标校录取位次数字更大 → 校更易 → 偏**保/垫**

### 分层参考（与 rank-adjustment 一致）

| 层级 | 目标校最低位次 vs 考生 |
|------|------------------------|
| 冲 | 低于考生 0–15%（数字更小） |
| 稳 | 偏差 ±5% |
| 保 | 高于考生 10–30%（数字更大） |
| 垫 | 高于考生 >30% |

### 使用注意

- 招生计划变化 >20% 时，位次参考**降置信度**，标注「未做计划调整」
- 合并批次、改规则年份的数据需降权或排除
- 单年「大小年」异常值不应单独作为冲/稳/保依据

## 常见退档原因

1. 总分达标但所报专业均无法录取，且不服从调剂
2. 体检不符合（色盲色弱、视力、身高等）
3. 单科成绩不满足招生章程
4. 语种、口试、面试等附加要求未满足
5. 投档比例 1:1.05 等导致的计划外退档（部分省份/院校）

## 批次与梯度策略

- **本科批**：冲稳保完整配置；注意批间不互斥省份的时间顺序
- **提前批**：通常顺序志愿或志愿数少，评估「是否值得占用」
- **专科批**：防滑档为主，保垫比例应更高
- **征集志愿**：作为报告附录提醒，不纳入主志愿表序号

## 数据核对清单

评估或推荐完成后，提醒用户核实：

- [ ] 目标院校当年《招生章程》
- [ ] 省教育考试院最新一分一段表
- [ ] 专业选科要求（省招办系统）
- [ ] 体检受限专业对照表
- [ ] 学费、中外合作、办学地点（是否异地校区）
- [ ] 大类招生分流规则

## 术语统一

全文统一使用：

- **位次**（不用「排名」混指不同含义）
- **投档** / **录取**（区分「投出」与「正式录取」）
- **专业组**（新高考，不用「专业包」）
- **服从调剂**（不用「服从分配」）
- **约 / 参考 / 区间**（描述薪资、位次、就业比例时必带，见 [anti-hallucination.md](shared/anti-hallucination.md)）

# Agent 交接包（推荐 → 评估）

> 推荐 Agent 结束、启动**评估 Agent（新会话）**时，只传递本模板 + 用户确认后的方案。**禁止**把整段推荐对话或全库文件带入评估会话。

与 [intake-8d.md](intake-8d.md)、[workflows/evaluation.md](../workflows/evaluation.md) 配合。

---

## 何时交接

| 场景 | 动作 |
|------|------|
| 推荐 Step9 通过，用户要「审核/看看有没有坑」 | 输出交接包 → **新开会话** @评估 Agent |
| 用户直接给志愿表 | **不经过**推荐，评估 Agent 独立启动 |
| 同一会话里已跑完推荐 | **不要**继续 load evaluation.md；改交接 |

---

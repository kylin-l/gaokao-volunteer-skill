# 反幻觉与表达规范

评估/推荐输出前**必须**遵守。与 [adi-assessment.md](adi-assessment.md) 配合：ADI ❌ 时默认不推荐。

## 五不准

| # | 不准 | 正确做法 |
|---|------|----------|
| 1 | 编造录取位次/分数线 | 查不到标「待核实」，宁缺毋滥 |
| 2 | 把旧年数据当当年 | 显式标注数据年份；勿用往年规则冒充当年 |
| 3 | 预测未来年份具体数值 | 「2027 年一本线约 XXX」禁止 |
| 4 | 给百分比录取概率 | 改为位次区间 + 冲/稳/保/垫档位 |
| 5 | 用聚合站替代官方源 | 二手源仅作线索，结论须回官方核验 |

## 表达禁令

| 禁止 | 替代 |
|------|------|
| 稳上 / 一定能录 / 大概率录取 | 根据往年数据，位次在该校录取范围内，可作冲/稳/保考虑 |
| 录取概率 XX% | 位次在往年区间 XX 位置，属冲/稳/保档 |
| 保证 / 担保 / 必录 | — |
| 捡漏 / 兜底神校 | — |
| 月薪 X 万（无区间无依据） | 同校同专业近年就业报告/实例：**约** X–Y k/月（标注来源与年份） |
| 「就业很好/一般」（无实例） | 中位数去向 + 具体岗位 + 收入区间 |

## 硬边界

- 缺**省份、年份、位次** → 不给具体候选，只输出 [templates/data-check.md](templates/data-check.md)
- 不替用户提交官方志愿系统
- 商业榜单、短视频、家长经验帖 → 线索，非结论依据
- 特殊类型（医学/公安/军校/艺术/定向/中外合作）→ 单独提醒限制条件
- **ADI ❌ 或任一维度 = 1** → 默认不得写入冲/稳/保主力；用户坚持须完成「书面确认」（见 adi-assessment）

## 谨慎原则

1. **先核实，后建议**：P0 信息不全时暂停，不猜测省份/位次
2. **先排斥，后喜好**：排斥项触发优先于「感兴趣」
3. **中位数，非顶流**：就业实例用普通毕业生，不用明星/极端案例
4. **标注不确定性**：计划调整未做、二手来源、ADI 待查 → 显式降置信度
5. **保留退路**：每条推荐说明转专业/考研/考公的大致难度，不粉饰
6. **不替决策**：给出依据与选项，最终填报由用户及家长确认

## 交付前自检

```
□ 每条位次/分数线有来源和年份？
□ 候选状态区分：可报 / 需核验 / 不建议？
□ 冲稳保仅为研究分层，非录取承诺？
□ 专业限制（选科/体检/单科/学费/校区）已逐项核对或标注待核验？
□ 每个专业方向已做 ADI 或标注「待补评」？
□ 排斥项已检查且无静默冲突？
□ ADI ❌ 项若保留，是否有用户书面确认记录？
□ 薪资/就业表述含「约/参考」及依据？
□ 报告含免责声明？
```

## 免责声明（报告末尾必附）

本方案/评估基于用户提供的信息与公开资料检索，仅供参考，**不构成录取承诺**。冲/稳/保/垫仅为基于往年位次的研究性分层，实际录取受当年计划、报考人数、政策变化等影响。请以省教育考试院、阳光高考及院校招生章程为准。建议与高中老师、招生办电话确认后再填报。

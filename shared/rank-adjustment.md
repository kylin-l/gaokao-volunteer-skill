# 位次匹配与计划调整

## 位次 vs 分数

| | 分数 | 位次 |
|--|------|------|
| 稳定性 | 年波动大 | 相对稳定 |
| 跨年可比 | 差 | **核心坐标** |

公式：`位次偏差率 = (目标最低位次 - 考生位次) / 考生位次`

| 层级 | 偏差率（参考） | 志愿占比参考 |
|------|----------------|--------------|
| 冲 | 目标校录取位次低于考生 0–15% | 20–30% |
| 稳 | 与考生位次偏差 ±5% | 40–50% |
| 保 | 目标校录取位次高于考生 10–30% | 25–30% |
| 垫 | 高于考生 >30% | 留 2–3 个 |

**数值示例**（考生位次 10000）：

| 目标校最低位次 | 偏差率 | 层级 |
|----------------|--------|------|
| 8500 | -15% | 冲 |
| 10000 | 0% | 稳 |
| 12500 | +25% | 保 |
| 14000 | +40% | 垫 |

详见 [../reference.md](../reference.md) 冲稳保换算说明。

仅看排名会忽略**计划增减**。某省本科计划今年比去年多 10% 时，同位次竞争力相当于去年更靠前。

```
等效去年位次 = 今年实际位次 × (去年本科招生总计划 / 今年本科招生总计划)
```

**示例**：去年计划 20 万，今年 22 万（+10%）  
考生今年排 100000 → 等效去年 ≈ 100000 × (200000/220000) ≈ **90909**

| 考生段位 | 计划变化影响 |
|----------|--------------|
| 前 1% | 极小 |
| 10–30% | 中等 |
| 30–70% | **关键** |
| 70% 以后 | 巨大 |

**无计划数据时**：标注「未做计划调整，待查省考试院招生计划」，仍可用裸位次分层但降置信度。

## 波动性与阈值

| 省份波动 | 冲稳保位次阈值 | 说明 |
|----------|----------------|------|
| 低（江苏等） | 偏紧 | 历史数据较稳 |
| 中 | 默认上表 | |
| 高（大小年明显） | 放宽 + 多看 3 年 | 单年数据降权 |

## 评估时必查

- [ ] 近 3 年最低录取**位次**（非仅分数）
- [ ] 当年招生计划 vs 去年变化 >20%？
- [ ] 新增/撤销专业、大类分流规则？
- [ ] 大小年：去年异常低/高是否回归？

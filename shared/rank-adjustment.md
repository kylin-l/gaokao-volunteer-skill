# 位次匹配与计划调整

## 报考年推算方向（必守）

**主流程**：锚定**当年考生位次**；将**历史院校投档**向前预测至报考年，再筛选。

```
预测报考年投档位次 = 去年投档位次 × K_score × (1 + δ_plan)
位次偏差率 = (预测报考年投档位次 - 考生当年位次) / 考生当年位次
```

| 做法 | 是否主流程 |
|------|------------|
| **2025 校投档 → 预测 2026 → 与 2026 考生位次比** | ✅ **是** |
| 2026 考生位次折回 2025 等效再比 | 仅心算/抽检 |

**禁止**：仅用去年裸投档排志愿序，再事后贴报考年预测数字。

**Step 3 出口**：须有一张**全量预测底表**（每志愿候选：去年投档、报考年预测、偏差、内部分层）；不全不准进入 Step 7 排序。

**官方投档公布后**：只替换「报考年预测」列；考生位次不变。

## 位次 vs 分数

| | 分数 | 位次 |
|--|------|------|
| 稳定性 | 年波动大 | 相对稳定 |
| 跨年可比 | 差 | **核心坐标** |

公式：`位次偏差率 = (目标最低位次 - 考生位次) / 考生位次`

| 层级 | 偏差率（参考） | 志愿占比参考 |
|------|----------------|--------------|
| 冲 | 目标校录取位次低于考生 0–15% | 20–30% |
| 稳 | 与考生位次偏差 ±5% | 40–50% |
| 保 | 目标校录取位次高于考生 10–30% | 25–30% |
| 垫 | 高于考生 >30% | 留 2–3 个 |

**数值示例**（考生位次 10000）：

| 目标校最低位次 | 偏差率 | 层级 |
|----------------|--------|------|
| 8500 | -15% | 冲 |
| 10000 | 0% | 稳 |
| 12500 | +25% | 保 |
| 14000 | +40% | 垫 |

详见 [../reference.md](../reference.md) 冲稳保换算说明。

仅看排名会忽略**计划增减**。某省本科计划今年比去年多 10% 时，同位次竞争力相当于去年更靠前。

```
等效去年位次 = 今年实际位次 × (去年本科招生总计划 / 今年本科招生总计划)
```

**示例**：去年计划 20 万，今年 22 万（+10%）  
考生今年排 100000 → 等效去年 ≈ 100000 × (200000/220000) ≈ **90909**

| 考生段位 | 计划变化影响 |
|----------|--------------|
| 前 1% | 极小 |
| 10–30% | 中等 |
| 30–70% | **关键** |
| 70% 以后 | 巨大 |

**无计划数据时**：标注「未做计划调整，待查省考试院招生计划」，仍可用裸位次分层但降置信度。

## 波动性与阈值

| 省份波动 | 冲稳保位次阈值 | 说明 |
|----------|----------------|------|
| 低（江苏等） | 偏紧 | 历史数据较稳 |
| 中 | 默认上表 | |
| 高（大小年明显） | 放宽 + 多看 3 年 | 单年数据降权 |

## 评估时必查

- [ ] 近 3 年最低录取**位次**（非仅分数）
- [ ] 当年招生计划 vs 去年变化 >20%？
- [ ] 新增/撤销专业、大类分流规则？
- [ ] 大小年：去年异常低/高是否回归？

## 标注层级时必做（合理性门禁）

给任一志愿标冲/稳/保/垫/探索前：

1. **计算** `位次偏差率 = (目标最低位次 - 考生位次) / 考生位次`
2. **对照**上表允许区间；超出 → 改层级或换校，**禁止**仅为「好听」标稳
3. **写入**候选矩阵：层级列与偏差率一致；异常须在「待核验」说明

整张表结构（保垫、梯度、调剂、2026 探索等）→ [structure-sanity-check.md](structure-sanity-check.md)。

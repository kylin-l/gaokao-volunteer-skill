# 高考体检与专业受限对照

依据教育部、卫生部、中国残联《普通高等学校招生体检工作指导意见》及院校招生章程**附加要求**。体检结论以**定点医院体检表**和**当年章程**为准，本表为志愿筛查线索。

与 [special-admissions.md](special-admissions.md)、[data-query.md](data-query.md) 配合。

---

## 一、体检结论三类

| 结论 | 含义 | 志愿动作 |
|------|------|----------|
| **患有某疾病，学校可以不予录取** | 见指导意见第二部分 | 相关院校可拒录，慎报 |
| **患有某疾病，不宜就读某类专业** | 见第三部分「不宜就读」 | 默认**不建议**报该类专业 |
| **患有某疾病，有关专业可不予录取** | 见第三部分「不予录取」 | 报了可能**退档** |

**院校可在章程中提出更严要求**（如航海视力 1.0），须**章程优先**。

---

## 二、常见硬性限制速查

### 视力

| 情况 | 常见受限专业（示例） |
|------|---------------------|
| 任何一眼**失明** | 不宜就读医学类各专业、工学、农学、理学、管理学等（见指导意见原文） |
| 任何一眼**低于 5.0** | 不宜就读**飞行、航海、消防、刑侦、轮机、运动训练**等 |
| 任何一眼**低于 4.8** | 不宜就读**轮机、运动训练、民族传统体育**等 |
| **色弱** | 不宜就读**化学、化工、制药、生物科学、公安技术、地理科学、医学**等 |
| **色盲** | 不宜就读**除色弱外**，还含**材料、环境、矿业、农学、医学、艺术、计算机**等（范围见原文） |
| **单色识别不全** | 不宜就读**计算机、化学、医学、艺术、金融、管理**等 |

**设计/美术类**：部分院校对色觉有要求，章程单列。

### 身高

| 情况 | 常见受限 |
|------|----------|
| 男生 **<170cm** | 部分**公安、外交、金融管理**等（以章程为准） |
| 女生 **<158cm** | 部分**外交、金融、空乘、师范**等 |
| 军警检 | **更严**，须军检标准 |

### 听力 / 口吃 / 疾病史

| 情况 | 常见受限 |
|------|----------|
| 两耳听力均在 **3 米以内** | 不宜就读**法学、外交、新闻、音乐**等 |
| **口吃** | 不宜就读**外交、新闻、师范**等 |
| **重症或难治性癫痫**等神经系统疾病 | 不宜就读**医学**等 |
| **心肌炎、胃十二指肠溃疡**等慢性病 | 不宜就读**法学、新闻、外交、化工、航海**等 |
| **传染病未愈**（肝功等） | 部分**食品、医学、师范**受限 |

## 三、专业类填报前体检自检清单

对志愿表中每个专业，逐项核对招生章程 + 本表：

```markdown
| 专业 | 色觉 | 视力 | 身高 | 听力 | 其他 | 结论 |
|------|------|------|------|------|------|------|
| 临床医学 | | | | | | 可报/不宜/待核验 |
```

- **不宜就读** → 评估报告标「结构风险」，默认建议替换
- **章程明确不予录取** → 评级 **F** 规则合规项

## 四、特殊类型体检

| 类型 | 体检 |
|------|------|
| 军校 / 武警 | **军检**（独立于普通体检） |
| 公安院校 | 公安类体检标准 |
| 飞行技术 | 招飞体检（非常严格） |
| 航海 / 轮机 | 视力、辨色力、无口吃等 |
| 师范 | 部分省要求无传染病等 |
| 医学 | 色盲色弱、肝功能等 |

---

## 五、数据从哪里查

1. 考生体检表「体检结论」栏
2. 省招办发布的**体检受限专业汇编**（部分省有）
3. 目标院校**招生章程**「体检要求」
4. 阳光高考院校库专业备注

**禁止**：凭记忆判断「应该没事」—— 退档不可挽回。

## 六、与家长沟通

| 家长说 | 回应 |
|--------|------|
| 「近视不深应该没事」 | 航海飞行轮机看**裸眼视力**；章程可能 4.8/5.0 |
| 「色弱想学计算机」 | 指导意见列计算机为不宜之一；部分校不招 |
| 「体检过了就能报」 | 普通体检过 ≠ 军检/招飞过 |

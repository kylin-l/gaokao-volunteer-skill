# 社会现实通俗指南

面向**对职场和社会了解有限**的高考生。用具体场景代替抽象概念，先建立认知，再谈志愿。

专业详解已拆分为分册（约 **100+ 方向**），本文件保留**通用原则与工具**。

---

## 使用原则

1. **先讲「一天怎么过」，再讲「好不好」**——学生更容易理解
2. **用中位数案例，不用明星案例**——「进投行」不是普通金融生的日常
3. **每个判断标注依据**：就业报告 / 行业周期 / 用户条件
4. **允许说「不适合你」**——比模糊鼓励更有用

## 二、五条出路，用实例说明

> **2026目录新增 / 风口专业**：必走 [new-major-guide.md](new-major-guide.md) 五步筛查 + 双路径；推荐卡片标 `2026目录新增`。详见 [examples.md](../examples.md) 实例 6–7。

### 1. 体制内（公务员 / 编制 / 国企）

**真实路径**：本科 → 应届身份考公/考编 → 可能多次尝试  
**例子**：汉语言文学本科 → 县级教育局科员，到手 4k–6k + 稳定  
**志愿影响**：院校层次影响**选调生资格**；专业影响**岗位表匹配**

### 2. 市场化就业（企业）

**真实路径**：本科 → 校招 → 1–3 年跳槽涨薪  
**例子**：电气本科 → 新能源企业设备工程师，3 年后约 12k/月（待核实）  
**志愿影响**：**城市实习机会** > 学校名气（同等层次下）

### 3. 技能型稳定（电网 / 护理 / 铁路）

**真实路径**：对口专业 → 校招/统考 → 长期稳定  
**例子**：华北电力 → 国网某省公司；西南交大 → 铁路局系统  
**志愿影响**：**行业院校**有时 > 综合排名

### 4. 深造换赛道（考研 / 留学）

**真实路径**：本科一般 → 考研 985/211 → 提升平台  
**例子**：普通二本 → 考研到 211 计算机 → 进中等规模 IT 公司  
**志愿影响**：选**保研率较高**或学习氛围好的学校作跳板

### 5. 创业 / 个体（窄路）

**真实路径**：口腔、设计、电商等——**手艺+客户**，学历是辅助  
**例子**：口腔本科 → 规培 → 多年后开诊所（需资本+执照）  
**志愿影响**：仅适合家庭有一定容错、本人有明确意愿者

## 三、如何帮「不知道自己要什么」的学生

不要问「你想学什么专业」。改用**具体情境选择题**：

### 情境题 A：周末更愿意做什么？

| 选项 | 可能方向（需结合成绩再定） |
|------|---------------------------|
| A 拆装东西、修电脑、做手工 | 工科、电气、机械 → [stem-engineering.md](social-reality/stem-engineering.md) |
| B 打游戏研究攻略、写点小代码 | 计算机 → [stem-it.md](social-reality/stem-it.md) |
| C 陪老人聊天、照顾小动物 | 护理、动医、师范 → 医卫/农林/教育分册 |
| D 看纪录片、写读后感 | 中文、历史、法学 → [education-humanities.md](social-reality/education-humanities.md) |
| E 都不喜欢，只想躺 | **先谈就业确定性**，勿追热门 |

### 情境题 B：最不能接受哪种工作？

| 排斥项 | 应排除的方向 |
|--------|-------------|
| 夜班 | 护理 ICU、临床急诊、铁路调度、部分生产岗 |
| 长期出差 | 土木现场、地质、部分销售岗 |
| 反复跟人扯皮 | 销售、客服、部分行政 |
| 坐不住办公室 | 纯 desk job 的会计/文员 |
| 数学物理痛苦 | 计算机核心、电气、AI、金融工程 |

### 情境题 C：四年后你最想要？

| 选择 | 志愿策略倾向 |
|------|-------------|
| 尽快挣钱养家 | 4 年制 + 就业确定性（护理、电气、软件） |
| 慢慢熬但后面稳 | 医学、师范、电网 |
| 还想读书 | 选保研率高、学科评估好的学校 |
| 先上岸再说 | 冲稳保 + 保专业可调 |

## 五、信息从哪里查（手把手）

| 想搞清楚什么 | 去哪查 | 看什么 |
|-------------|--------|--------|
| 这个专业毕业去哪 | 目标院校官网 → **就业质量报告** | 「单位性质」「主要就业行业」 |
| 真实工资 | 同上 + 招聘网站（参考） | **中位数区间**，不看「最高」 |
| 学什么、累不累 | 招生网 → **培养方案** | 核心课程列表 |
| 能不能转专业 | 教务处网站 | 转专业条件（成功率常需问学长） |
| 行业是不是在走下坡 | 搜「[行业] 招聘 [当年]」 | 多条来源交叉看 |

**实例：如何读就业报告**

某学院写「就业率 95%」→ 追问灵活就业算不算 → 看签约单位是头部企业还是本地小公司 → 看薪酬是税前还是含补贴表述。

## 六、2030 视角（入学 4 年后）

| 方向 | 2030 | 白话 | 分册 |
|------|------|------|------|
| 电气 / 电网 / 新能源 | 🟢 | AI 要用电 | stem-engineering |
| 临床 / 护理 / 康复 | 🟢 | 执照+照护难替代 | medicine-health |
| 计算机 / 软件 | 🟡 | 会 AI 工具更强，只会基础写码更难 | stem-it |
| 微电子 / 部分材料 | 🟢/🟡 | 战略产业但院校分化大 | stem-engineering |
| 会计 / 基础行政 | 🔴 | AI 做账写材料 | business |
| 土木（无资源） | 🔴 | 周期底部 | stem-engineering |
| 普通金融 / 工商营销 | 🔴 | 基础岗压缩 | business |
| 新闻 / 传统编导 | 🔴 | 岗位萎缩 | education-humanities |
| 师范（基层） | 🟢/🟡 | 编制竞争但需求在 | education-humanities |
| 农学 / 动医（宠物向） | 🟡/🟢 | 看细分赛道 | agri-vet |

**跟学生说**：选专业不是选「现在哪个名字好听」，是选 **2029 年 6 月你拿着简历去找工作** 时，哪个方向还愿意要你。

## 七、窄门专业解锁检查

部分专业**不是分数够就能读、读了就能就业**。推荐/评估 Step 4–5 命中下表时，须逐项核对；未解锁 → ADI 相关维度 ≤2 或整体 ❌，默认不推荐作本科主力。

| 窄门类型 | 典型专业 | 解锁条件（须用户明确具备） | 未解锁时的路径 B |
|----------|----------|---------------------------|------------------|
| AI/算法 | 计算机、AI、具身智能、智能计算 | 数学接近满分或竞赛/项目；家庭支持读研 | 自动化、电子信息 |
| 临床医学 | 临床、口腔 | 接受 5+3/长学制；不怕夜班；体检过关 | 护理、康复、公卫 |
| 金融顶尖 | 金融、量化 | 顶尖校+数学强+实习资源；接受从基层做起 | 会计、统计 |
| 艺术/design | 设计、数媒、播音 | 作品集/统考过硬；接受改稿加班 | 数媒技术、师范美术 |
| 军警/fire | 公安、军校、森林消防 | 体检政审体能全过；真实意愿 | 法学、应急管理 |
| 风口 2026 | 具身智能、脑机、低空首招 | 强校母体+读研意愿；非短视频种草 | 见 [new-major-guide.md](new-major-guide.md) 路径 B |

**用法**：在方向包写「窄门解锁：✅/❌/待确认」及依据。完整对话见 [examples.md](../examples.md) **实例 6**。

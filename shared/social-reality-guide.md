# 社会现实通俗指南

面向**对职场和社会了解有限**的高考生。用具体场景代替抽象概念，先建立认知，再谈志愿。

## 使用原则

1. **先讲「一天怎么过」，再讲「好不好」**——学生更容易理解
2. **用中位数案例，不用明星案例**——「进投行」不是普通金融生的日常
3. **每个判断标注依据**：就业报告 / 行业周期 / 用户条件
4. **允许说「不适合你」**——比模糊鼓励更有用

## 一、热门专业：你以为 vs 实际上

### 计算机 / 软件工程

| 你以为 | 实际上（普通毕业生中位数） |
|--------|---------------------------|
| 天天写代码、做 APP | 大量时间在改 bug、写文档、开会对接需求 |
| 毕业进大厂拿高薪 | 进中小公司、外包、政企信息化岗更常见 |
| AI 来了更吃香 | **入门层**（简单页面、基础 CRUD）正被 AI 压缩；**架构/AI 工程**仍缺人 |
| 坐办公室很轻松 | 项目上线前加班常见，但比工地/夜班温和 |

**一天快照（工作 3 年的后端工程师，二线城市）**：
> 9:30 站会 → 10:00 改接口 bug → 14:00 和产品经理吵需求 → 16:00 写单元测试 → 18:30 下班，上线前可能到 22:00。  
> 月薪到手约 8k–12k（非顶尖城市非大厂）。

**适合谁**：数学逻辑不差、能持续自学、接受「技术更新快」、不排斥电脑久坐。  
**不适合谁**：数学很吃力、讨厌反复调试、希望「学完就稳定不变」。

### 金融学（非顶尖 985）

| 你以为 | 实际上（普通院校金融本科） |
|--------|---------------------------|
| 穿西装、在陆家嘴做分析师 | 多数进**银行网点**做柜员/理财经理/贷款客户经理 |
| 研究股票、财富自由 | 卖理财 KPI、拉存款、客户投诉处理 |
| 专业名字好听、体面 | 资源依赖极高——没背景很难进核心岗 |
| AI 时代更智能 | AI 正在替代**数据整理、报告撰写**——正是新人主要工作 |

**一天快照（某城商行网点理财经理，工作 2 年）**：
> 8:30 晨会 → 9:00 接待客户办业务 → 全天穿插推销理财/保险 → 17:30 填业绩表 → 月末冲 KPI 常加班。  
> 月薪到手约 5k–8k + 波动绩效（看完成率）。

**具体建议**：
- 清北复交级别 + 家庭有资源 → 可认真考虑
- 普通一本金融 → **建议改方向**：统计学、计算机、会计学（叠加 CPA）更实在
- 家长说「学金融有前途」→ 用上面快照解释「前途分层次」

### 临床医学

| 你以为 | 实际上 |
|--------|--------|
| 白大褂、受人尊敬 | 前 8–10 年**收入低、强度大**（规培期月薪可能 2k–4k） |
| 毕业当医生 | 本科几乎不够，多数路径：5 年本科 + 3 年规培 + 可能专培 |
| 女生不适合 | **错误**。内科、儿科、皮肤、妇产等女生很多；关键是能否接受长学制 |
| AI 会替代医生 | 辅助诊断在进步，但**执照+操作+医患沟通**短期难替代 |

**一天快照（规培第 2 年，内科）**：
> 6:30 到病房查房 → 8:00 跟上级收新病人 → 白天写病历、开医嘱 → 经常 20:00 后下班 → 周末有时值班。  
> 规培到手约 3k–5k/月（因城市而异）。

**选校关键**：**附属三甲医院** > 学校综合排名。  
例：温州医科大学（有附院）> 同分数段无附院综合大学。

### 电气工程及其自动化

| 你以为 | 实际上 |
|--------|--------|
| 修电线杆、爬高危险 | 大量岗位在**电网调度中心、设计院、工厂自动化**——办公室+现场结合 |
| 传统夕阳专业 | AI 时代**数据中心用电、新能源、智能电网**需求上升 |
| 只能去国家电网 | 也可去电厂、地铁、制造业、新能源企业 |

**一天快照（地级市国家电网，工作 4 年）**：
> 8:30 看调度数据/巡检计划 → 10:00 现场或远程监控 → 14:00 写报告 → 17:00 下班，故障时抢修。  
> 到手约 7k–11k/月（含稳定福利，因地区差异大）。

**适合谁**：物理不错、求稳定、接受体制内节奏、家庭资源一般。

---

### 土木工程（2026 时点）

| 你以为 | 实际上 |
|--------|--------|
| 盖房子、国家永远需要 | 房地产深度调整，**设计院裁员、项目减少**是现实 |
| 男生好就业 | 就业面在缩，很多毕业生**转码、考公、转行** |
| 收入高 | 以前高，现在**起薪和增长都弱于电气/计算机** |

**具体实例**：2019 年入学土木 → 2023 毕业，不少同学未对口就业或考研换方向。  
**不是绝对不能报**：若家族有建筑行业资源、或目标注册建筑师且能接受周期，可个案讨论。普通家庭普通分数**默认降级推荐**。

### 护理学

| 你以为 | 实际上 |
|--------|--------|
| 伺候人、没地位 | 三甲 ICU/手术室护士**技能门槛高**，收入不一定低于普通白领 |
| 女生专属 | **男生是隐藏优势**：男护在 ICU/急诊/手术室极抢手 |
| 轻松 | **三班倒、夜班**是常态，体力+心理压力大 |

**一天快照（三甲 ICU 护士，工作 3 年）**：
> 7:00 交接班 → 8:00–16:00 或 16:00–24:00 班 → 测生命体征、执行医嘱、家属沟通 → 夜班后补觉。  
> 到手约 6k–10k/月（一线城市三甲更高）。

**适合谁**：能接受夜班、细心、求就业确定性；男生同分段常是**性价比之选**。

### 法学（非五院四系）

| 你以为 | 实际上 |
|--------|--------|
| 律师法庭辩论很酷 | 新人大量做**合同整理、案例检索、文书起草**——正被 AI 压缩 |
| 当法官检察官 | 需过**法考** + 考公/竞聘，竞争极烈 |
| 学法学好当公务员 | 汉语言文学师范**考公面更广**；法学岗位并非最多 |

**五院四系**：中国政法、西南政法、西北政法、华东政法、中南财经政法 + 北大/人大/武大/吉大法学——**普通院校法学需显著降级预期**。

---

### 新闻学 / 传播学

| 你以为 | 实际上 |
|--------|--------|
| 当记者、采访名人 | 行业萎缩 + AI 写稿，**对口岗位极少** |
| 做自媒体也算对口 | 自媒体**不依赖学历**，四年新闻学优势不明显 |

**具体建议**：有强烈表达欲 → 可考虑汉语言、广告、新传研究生路径；求就业 → **默认不推荐**本科新闻作为主方向。

## 二、五条出路，用实例说明

### 1. 体制内（公务员 / 编制 / 国企）

**真实路径**：本科 → 应届身份考公/考编 → 可能多次尝试  
**例子**：汉语言文学本科 → 县级教育局科员，到手 4k–6k + 稳定  
**志愿影响**：院校层次影响**选调生资格**（部分 985/211 有定向选调）；专业影响**岗位表匹配**（中文、法学、财会岗多）

### 2. 市场化就业（企业）

**真实路径**：本科 → 校招 → 1–3 年跳槽涨薪  
**例子**：电气本科 → 某新能源企业设备工程师，3 年后 12k/月  
**志愿影响**：**城市实习机会** > 学校名气（同等层次下）

### 3. 技能型稳定（电网 / 护理 / 铁路）

**真实路径**：对口专业 → 校招/统考 → 长期稳定  
**例子**：华北电力 → 国网某省公司；西南交大 → 铁路局系统  
**志愿影响**：**行业院校**有时 > 综合排名

### 4. 深造换赛道（考研 / 留学）

**真实路径**：本科一般 → 考研 985/211 → 提升平台  
**例子**：普通二本 → 考研到 211 计算机 → 进中等规模 IT 公司  
**志愿影响**：选**保研率较高**或**学习氛围好**的学校作跳板

### 5. 创业 / 个体（窄路）

**真实路径**：口腔、设计、电商等——**手艺+客户**，学历是辅助  
**例子**：口腔本科 → 规培 → 3 年后开诊所（需资本+执照）  
**志愿影响**：仅适合家庭有一定容错、本人有明确意愿者

## 三、如何帮「不知道自己要什么」的学生

不要问「你想学什么专业」（他们答不上来）。改用**具体情境选择题**：

### 情境题 A：周末更愿意做什么？

| 选项 | 可能方向（需结合成绩再定） |
|------|---------------------------|
| A 拆装东西、修电脑、做手工 | 工科、电气、机械 |
| B 打游戏研究攻略、写点小代码 | 计算机（需看数学） |
| C 陪老人聊天、照顾小动物 | 护理、动物医学、师范 |
| D 看纪录片、写读后感 | 中文、历史、法学 |
| E 都不喜欢，只想躺 | **先谈就业确定性**，勿追热门 |

### 情境题 B：最不能接受哪种工作？

| 排斥项 | 应排除的方向 |
|--------|-------------|
| 夜班 | 护理 ICU、临床急诊、铁路调度 |
| 长期出差 | 土木现场、部分销售岗 |
| 反复跟人扯皮 | 销售、客服、部分行政 |
| 坐不住办公室 | 纯 desk job 的会计/文员 |
| 数学物理痛苦 | 计算机核心、电气、AI |

### 情境题 C：四年后你最想要？

| 选择 | 志愿策略倾向 |
|------|-------------|
| 尽快挣钱养家 | 4 年制 + 就业确定性（护理、电气、软件） |
| 慢慢熬但后面稳 | 医学、师范、电网 |
| 还想读书 | 选保研率高、学科评估好的学校 |
| 先上岸再说 | 冲稳保 + 保专业可调 |

## 四、给家长看的「一句话对照表」

| 家长常说 | 你可以这样回应（带实例） |
|----------|-------------------------|
| 「学金融有前途」 | 普通金融多数进银行网点，不是投行；例：2 年理财经理 5k–8k |
| 「必须上 985」 | 若目标在上海就业，上海大学有时比偏远 985 实习资源更好 |
| 「计算机永远热门」 | 入门编程岗在变少，要会 AI 工具+复杂问题；数学弱慎选 |
| 「医生越老越值钱」 | 对，但前 10 年很苦；规培 3k–5k，要家庭能撑 |
| 「学师范没出路」 | 考编是窄门但稳定；语文 140 分是考编优势 |
| 「男生学护理丢人」 | 男护 ICU 抢招，同分数就业常优于普通工科 |

## 五、信息从哪里查（手把手）

| 想搞清楚什么 | 去哪查 | 看什么 |
|-------------|--------|--------|
| 这个专业毕业去哪 | 目标院校官网 → **就业质量报告** | 「单位性质」「主要就业行业」表格 |
| 真实工资 | 同上报告 + 招聘网站（作参考） | 看**中位数区间**，不看「最高」 |
| 学什么、累不累 | 院校招生网 → 培养方案 | 核心课程列表 |
| 能不能转专业 | 教务处网站 | 转专业条件、成功率（往往不写，需问学长） |
| 行业是不是在走下坡 | 搜「[行业] 招聘 2025 2026」 | 多条来源交叉看 |

**实例：如何读就业报告**

某大学计算机学院就业报告写：「就业率 95%」  
→ 追问：灵活就业算不算？对口 IT 多少？  
→ 看「签约单位名称」：是腾讯还是本地小型软件公司？  
→ 看「平均薪酬」：是税前还是含公积金表述？

## 六、2030 视角（入学 4 年后）

| 方向 | 2030 大致判断 | 给学生的白话 |
|------|--------------|-------------|
| 电气 / 电网 | 🟢 受益 | AI 要用电，这行更缺人 |
| 临床 / 护理 | 🟢 稳 | 机器替不了打针陪床 |
| 计算机 | 🟡 分层 | 会 AI 工具的更强，只会基础写码的更难 |
| 会计 / 行政 | 🔴 受压 | 做账、写材料 AI 在做 |
| 土木 | 🔴 周期底部 | 除非有家族资源，慎选 |
| 普通金融 | 🔴 极高压力 | 底层岗位在缩 |

**跟学生说**：选专业不是选「现在哪个名字好听」，是选 **2029 年 6 月你拿着简历去找工作** 时，哪个方向还愿意要你。

> 考生：{省} {年份} {科类} {分数} / 位次 **{N}**  
> **校专综评**：`字母｜校…｜专…｜业…｜匹…` 四段合成  
> **前景**：发展·就业·AI — 详见同目录产出 `11-major-outlook.md`（模板：[major-outlook.md](./major-outlook.md)）

### 校专综评读法

| 段 | 内容 |
|----|------|
| **字母** | S/A/B+/B/C — 学校+专业+匹配综合档 |
| **校** | 层次 + 软科总榜区间 + 行业地位 |
| **专** | **本科专业实力**（软科 2024 等）+ 对口学科评估；标注够硬/很硬/中等/偏弱 |
| **业** | 就业认可度（行业口碑、属地绑定） |
| **匹** | 对你：ADI / 地域④③②① / 家长 / 位次层级 |

**填写规范**：

- **院校名称用全称**，禁止仅写缩写（如「北邮」→「北京邮电大学」）
- **校专综评**四段缺一不可；字母为综合档，与四段内容一致
- **推荐依据**写此序号、此层级的填报理由（锚点、双轨、地域、家长备选等）
- **前景**格式：`{发展标签}·{就业标签}·{AI符号}{风险}`，如 `增量·极强·🟡中`
- 组内专业按 **1→6 录取优先级**排列；排斥项专业放末位或注明禁填
- 同校不同专业组须分行，**勿混组**（冲/保锚点须标注组号）
- 组线≠专业线时，在表下脚注 `†` 并说明须单独核对

## 列说明

| 列 | 含义 |
|----|------|
| **2026预测** | 模型估算位次；填报前须核对官方计划与当年线 |
| **偏差** | 相对考生位次的偏差率；负=校更难（冲），正=校更易（保） |
| **校专综评** | 字母 + 校/专/业/匹四段；字母为综合档 |
| **推荐依据** | 此序号、此层级的具体理由（填报决策用） |
| **前景** | 发展·就业·AI 替代 — 详见 `11-major-outlook.md` |
| **组内专业 1→6** | 组内志愿顺序；服从调剂时实际录取可能为组内其他专业 |

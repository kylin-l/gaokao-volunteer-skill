## 一、读标说明

### 发展趋势（5–10 年）

| 标签 | 含义 |
|------|------|
| **增量** | 需求仍扩张，但内部分化（要能力升级） |
| **稳增** | 需求稳定偏增，偏「数据+业务」复合 |
| **绿电** | 新能源、智能电网、储能带动 |
| **国产替代** | 芯片、通信、工业软件政策红利 |
| **分化** | 头部吃饱、普通院校要靠实习/考研突围 |
| **平稳** | 省内体制/行业系统为主，涨幅有限 |

### 就业（本考生位次段，看**中位数**）

| 标签 | 含义 |
|------|------|
| **极强** | 行业校招第一梯队，实习资源好 |
| **强** | 一线/强二线好找对口岗 |
| **中强** | 要实习，名校牌更重要 |
| **中** | 属地/省会够用，全国竞争力一般 |
| **稳** | 国网/电网/国企系统，波动小 |

### AI 替代风险（本科常见去向）

| 符号 | 风险 | 典型受影响工作 | 相对安全方向 |
|------|------|----------------|--------------|
| 🟢 **低** | 难被整体替代 | — | 现场运维、电网调度、嵌入式、网安、芯片 |
| 🟡 **中** | 工具增强，初级岗压缩 | 简单页面、基础报表、重复测试 | 系统架构、业务理解、算法+领域 |
| 🟠 **较高** | 大量事务性工作可自动化 | 基础数据分析、初级会计审核 | 风控建模、合规、高端统计 |
| 🔴 **高** | 本科对口岗易被大幅压缩 | — | — |

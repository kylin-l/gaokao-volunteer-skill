# 真实数据查询

查位次/分数线时**优先官方源或 WebSearch 核验**，禁止 LLM 编造。

## 方式一：省考试院官网（首选）

1. 打开 [data/official-source-index.json](data/official-source-index.json) 找目标省 URL
2. 查当年**一分一段表**、**批次线**、**招生计划**
3. 每条记录标注：来源名 + URL + 年份 + 科类

常见栏目：「信息公开」→「一分一段表」「批次线」「招生计划」。

## 方式二：WebSearch 辅助

无直接链接或页面结构变化时，用精确关键词搜索后**回官网核验**：

```
[省份] [年份] 高考 一分一段表 [科类]
[省份] [年份] 高考 批次线 [科类]
[院校全称] [年份] 招生章程
[院校] [年份] 就业质量报告
```

优先采用省考试院、阳光高考、院校官网结果；二手平台数据须交叉核对。

## 查询顺序（评估/推荐通用）

```
1. 一分一段 → 确定位次（只有分数时必做）
2. 批次线 → 判断批次可行域
3. 目标校近年投档最低位次（近 3 年）
4. 当年招生计划变化
5. 招生章程 → 选科/体检/学费/校区
```

# 官方信息源优先级

## 优先级

| 级 | 来源 | 用途 |
|----|------|------|
| 1 | **省教育考试院/招生考试机构** | 一分一段、批次线、投档规则、招生计划、填报入口 |
| 2 | **阳光高考 / 阳光志愿** | 政策、院校库、专业库、招生章程索引 |
| 3 | **高校招生官网** | 章程、专业限制、学费、转专业、校区 |
| 4 | **当年招生计划 + 往年录取统计** | 计划数、往年位次（须标年份） |
| 5 | 权威媒体、商业平台 | **仅线索**，须回到 1–3 核验 |

## 国家级入口

- 阳光高考：https://gaokao.chsi.com.cn/
- 阳光志愿：https://gaokao.chsi.com.cn/zyck/
- 院校库：https://gaokao.chsi.com.cn/sch/
- 专业库：https://gaokao.chsi.com.cn/zyk/
- 31 省招办索引：https://gaokao.chsi.com.cn/gkxx/zc/ss/202007/20200716/1542552579.html

各省考试院 URL 见 [data/official-source-index.json](data/official-source-index.json)。

## 核验规则

- **年份一致**：2026 任务不用 2025 政策/计划冒充
- **省份一致**：不跨省套用批次与投档规则
- **位次优先于分数**：只有分数时先查一分一段
- **计划优先于往年位次**：扩招/缩招会偏移匹配
- **限制逐项查**：选科、体检、语种、单科、性别、校区、学费、中外合作、专项资格
- **冲突标「需核验」**：写明冲突来源与下一步（招生办电话等）

## 候选状态

| 状态 | 含义 |
|------|------|
| 可报 | 必要条件已核验，风险主要来自竞争与排序 |
| 需核验 | 信息不全、来源冲突、计划未确认 |
| 不建议 | 硬性条件不满足、位次差距过大 |

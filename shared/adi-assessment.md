# ADI 四维专业匹配评估

用于**筛选专业方向**和**校验志愿表中的专业选择**。推荐 Step 6、评估 Step 4 使用。

与 [intake-8d.md](intake-8d.md)（硬信息）、[guided-discovery.md](guided-discovery.md)（情境引导）分工：本文件做**结构化打分**。

## 四维定义

| 维度 | 英文 | 问什么 | 1 分 | 3 分 | 5 分 |
|------|------|--------|------|------|------|
| **A 能力** | Ability | 学得下来吗 | 核心课明显吃力（如数学弱报计算机） | 能跟但需额外努力 | 学科优势匹配 |
| **D 需求** | Demand | 毕业时有市场吗 | 2030 🔴 且家庭容错低 | 🟡 或数据不全 | 🟢 中位数去向清晰 |
| **I 兴趣** | Interest | 愿意干这行吗 | 排斥项触发 / 一天快照无法接受 | 中性，可接受 | 明确喜欢工作类型 |
| **R 现实** | Reality | 家庭/身体/地域允许吗 | 学费/体检/地域硬性冲突 | 有代价但可协商 | 完全匹配约束 |

## 问卷（每个候选专业填一次）

### A 能力

```
1. 该专业核心 4 门课中，有几门对应你的强项？
   □ 0–1 门 → A=1–2  □ 2 门 → A=3  □ 3–4 门 → A=4–5
2. 最弱一门能否接受连续学 4 年？（如高数/化学/手绘）
   □ 不能 → A≤2  □ 勉强 → A=3  □ 可以 → A≥4
```

### D 需求

```
3. 中位数毕业生主要去向是否明确？（查就业质量报告）
   □ 说不清 → D≤2  □ 有数据 → D=3–4  □ 路径清晰 → D=5
   **2026目录新增**（《目录（2026年）》/首招）：无就业报告 → D≤2；超常布局 AI 类本科 → D≤3。见 [new-major-guide.md](new-major-guide.md)
4. 2030 年（本科毕业年）行业趋势？
   □ 🔴 → D≤2  □ 🟡 → D=3  □ 🟢 → D=4–5
```

### I 兴趣

```
5. [social-reality-guide](social-reality-guide.md) 及 [social-reality/](social-reality/README.md) 对应分册「一天快照」能否接受？
   □ 不能接受 → I≤2  □ 凑合 → I=3  □ 能接受 → I≥4
6. 排斥项（夜班/出差/销售/数学痛苦等）是否触发？
   □ 触发 → I=1  □ 无 → 继续
```

### R 现实

```
7. 学费/学制（5+3 医学等）家庭能否承担？
   □ 不能 → R≤2  □ 勉强 → R=3  □ 可以 → R≥4
8. 体检/选科/单科要求是否满足？
   □ 不满足 → R=1（硬性排除）  □ 待核验 → R≤2  □ 满足 → R≥4
9. 就业地域与家庭期望是否冲突？
   □ 严重冲突 → R≤2  □ 部分 → R=3  □ 一致 → R≥4
```

## 乘法模型（综合判定）

```
ADI综合 = A × D × I × R   （各维度 1–5 整数）
几何均值 = (A × D × I × R) ^ 0.25
```

| ADI综合 | 几何均值 | 判定 | 动作 |
|---------|----------|------|------|
| ≥256 | ≥4.0 | ✅ 强匹配 | 可作稳/冲主力 |
| 81–255 | 3.0–3.9 | 🟡 可接受 | 需说明代价，作稳/保 |
| 16–80 | 2.0–2.9 | ⚠️ 勉强 | 仅作冲或备选，必须写风险 |
| ≤15 | <2.0 | ❌ 不匹配 | 默认不推荐；用户坚持须书面确认 |

**任一维度 = 1** → 直接 ❌，不论乘积（硬性否决）。

## 用户坚持保留（ADI ❌ 时）

默认不得作为冲/稳/保主力。若用户或家长仍要求保留，**必须**在报告中记录：

```markdown
### 用户确认（ADI ❌ 项）

- 专业：[名称]
- ADI 判定：❌（[A/D/I/R]=1 的原因）
- 已告知风险：[具体风险，如数学吃力、2030 🔴、排斥项冲突]
- 用户回复：「已知风险，仍希望保留」
- 建议位置：仅作冲/备选，不得占稳/保主力
```

未获明确确认 → 不得写入推荐方案主力志愿。

## 与排斥项的关系

排斥项优先级 **高于** I 维度喜好：

| 情况 | 处理 |
|------|------|
| 用户说「喜欢」但排斥项触发 | I=1，ADI 否决 |
| 家长喜欢，学生 I≤2 | 方向条目中分列家庭/个人 ADI |
| 性别相关 | 仅在有公开供需数据时微调 D，不做刻板能力判断 |

## 输出格式（嵌入方向全景条目 / 评估报告）

```markdown
### ADI 评估：[专业名]

| 维度 | 得分 | 依据 |
|------|------|------|
| A 能力 | 4 | 物理强，高数中等 |
| D 需求 | 3 | 2030 🟡，就业报告待查 |
| I 兴趣 | 4 | 接受一天快照描述 |
| R 现实 | 5 | 学费/地域/体检均 OK |

**综合**：4×3×4×5 = 240（几何均值 3.46）→ 🟡 可接受
**主要风险**：行业周期不确定，建议查 XX 校就业质量报告
```

## 快速筛查（时间紧时）

只问 4 句，每句对应一维：

1. **A**：「这个专业最难的几门课，你哪几门会痛苦？」
2. **D**：「普通毕业生（不是学霸）大多去哪？2030 这行怎么样？」
3. **I**：「能接受 [一天快照] 这种日常吗？」
4. **R**：「学费、地域、体检、父母期望，有没有硬冲突？」

任一答案触发否决条件 → 不进入冲稳保清单。

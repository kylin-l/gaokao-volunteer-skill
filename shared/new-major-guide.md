# 2026 目录新增专业推荐指南

> **目录年份**：`2026` — 依据《普通高等学校本科专业目录（2026年）》（教高函〔2026〕2号）。  
> 输出时专业名须带前缀 **`2026目录新增`**（或表格列 `2026新增`）。次年招生季须核对教育部最新目录并更新年份。

面向**2026 目录新增、首次招生、超常布局**类本科专业。推荐 Step 5、评估 Step 4 涉及时**必读**。

与 [official-sources.md](official-sources.md)（核验源）、[adi-assessment.md](adi-assessment.md)（去留）、[social-reality-guide.md](social-reality-guide.md)（通俗对照）配合使用。

---

## 一、何时触发

满足任一即按本指南处理：

| 信号 | 示例 |
|------|------|
| 用户在问「新专业能不能报」 | 2026目录新增：具身智能、低空经济与管理 |
| 志愿表/方向含 **《本科专业目录（2026年）》新增 38 专业** | 见第三节速查 + [new-majors-2026.md](social-reality/new-majors-2026.md) |
| 专业名含 AI/智能/低空/数字/交叉 等且 **无往年录取位次** | 须标 `2026新增/首招` |
| 用户因短视频/风口种草 | 「具身智能年薪 30 万」 |
| 阳光高考专业库标注为近年新增 | 以当年目录为准 |

**不触发**：仅是「特设专业 T」但已招生多年、有稳定位次数据者——按普通专业处理。

## 二、五步筛查（推荐必做）

```
① 目录核验 → ② 本省招生核验 → ③ 母体学科核验 → ④ ADI 加严 → ⑤ 双路径输出
```

### ① 目录核验

- 查 [本科专业目录（2026）PDF](http://www.moe.gov.cn/srcsite/A08/moe_1034/s3882/202604/W020260427440749576927.pdf) 或 [教高函〔2026〕2号](https://www.gov.cn/zhengce/zhengceku/202604/content_7067247.htm)
- 记录：**专业代码、学位门类**（工学/管理学等）、是否 TK 国家控布点
- 区分：**目录新增** vs **交叉学科门类划入**（如未来机器人原在工学）

### ② 本省招生核验（硬性）

| 检查项 | 未通过则 |
|--------|----------|
| 省考试院**当年招生计划**有该校该专业 | 状态 = **需核验**，不得写入可报主力 |
| 招生章程：选科、体检、语种、学费、校区 | 冲突 → **不建议** |
| 近 3 年录取位次 | **无数据 → 不得标冲/稳/保**，仅「探索/待核验」 |

数据入口：[阳光高考专业库](https://gaokao.chsi.com.cn/zyk/) · 省考试院 · 院校招生网

### ③ 母体学科核验（2026目录新增核心）

新专业质量取决于**依托学院与既有学科**，不是专业名字。

| 2026目录新增类型 | 必问母体学科 | 弱母体时的动作 |
|------------|--------------|----------------|
| 具身智能 / 脑机接口 / 农业机器人 | 自动化、计算机、机械、控制 | 降级为传统工科推荐 |
| 低空经济与管理 / 低空安全管理 | 交通运输、航空航天、管理学 | 查是否真有空域/仿真平台 |
| 数字金融 / 数字贸易 / 商业人工智能 | 经济、金融、计算机、统计 | 普通院校优先会计/统计/计算机 |
| 生物制造 | 生物工程、化工、发酵 | 无实验室链条则谨慎 |
| 能源科学与工程 / 深地科学与工程 | 能动、地质、矿业 | 看行业院校属性 |
| 心理、脑与认知科学 | 心理学、神经科学、认知科学 | 非强校慎作唯一方向 |
| 数字文旅 / 艺术治疗 | 旅游管理、设计、心理 | 偏应用，看实习基地 |

**操作**：查院系官网 → 培养方案 → 核心课程是否 70% 以上来自成熟学科。

### ④ ADI 加严规则

在 [adi-assessment.md](adi-assessment.md) 基础上叠加：

| 维度 | 2026目录新增默认 | 升级条件 |
|------|------------|----------|
| **D 需求** | 无对口就业报告 → **D≤2** | 查到该校该方向就业质量报告或行业校招数据 → D=3–4 |
| **D 需求** | 超常布局战略专业（具身智能等） | 本科路径默认 D≤3（岗位多要求硕士） |
| **I 兴趣** | 因「名字酷/短视频」而喜欢 | 须完成一天快照；否则 **I≤2** |
| **A 能力** | AI 交叉类 | 数学/物理弱 → **A≤2**，推荐传统替代 |
| **R 现实** | 首招无位次 | 不得作家庭唯一保档 → **R≤3** 若家长押注全部希望 |

**层级限制（默认）**：

| ADI 判定 | 2026目录新增在志愿表中的位置 |
|----------|------------------------|
| ✅ 强匹配 | 可作 **冲**；有 1 年位次后方可作稳 |
| 🟡 可接受 | **冲 / 探索**；稳档须另有传统对口专业 |
| ⚠️ 勉强 | 仅 **冲** 或 2026 **方向条目**备选；须写替代路径 |
| ❌ | 不推荐；用户坚持走书面确认 |

**首招无位次**：该志愿层级列填 **「探索」**，候选矩阵状态 **需核验**，禁止写「稳上」。

### ⑤ 双路径输出（强制）

每个 **2026目录新增** **方向条目**必须同时给出：

```markdown
### 路径 A（2026目录新增）：[校+专业名]
- 招生核验：✅/需核验
- 母体学科：[学院/学科评估/平台]
- ADI：…
- 适合如果：…
- 主要风险：无位次 / 培养方案未验证 / 就业路径偏研究或硕士

### 路径 B（稳妥替代）：[校+传统专业名]
- 对应关系：如 具身智能 → 自动化/机器人工程；数字金融 → 金融学+辅修计算机
- ADI：…
- 为何更稳：有位次、有就业报告、考研可切入路径 A 方向
```

**默认策略**（除非用户 ADI 全 ✅ 且强校强母体）：

> 本科走 **路径 B**，研究生/实习阶段再切入 2026 目录新增方向。

## 三、2026 目录新增 38 专业 · 推荐替代速查

> 名单以教育部 PDF 为准；下列「稳妥替代」供推荐排序，非否定 2026 目录新增专业。

| 集群 | 新增专业（节选） | 本科稳妥替代（同校优先） | 2030 | 备注 |
|------|------------------|--------------------------|------|------|
| **AI 与交叉** | 具身智能、脑机科学与技术、语言智能、商业人工智能、智能计算 | 自动化、计算机、电子信息、机器人工程 | 🟡 | 岗位多要求硕士；见 [examples.md](../examples.md) 实例 6 |
| **低空经济** | 低空经济与管理、低空安全管理 | 交通运输、航空航天、自动化 | 🟡 | 与 2024 低空技术与工程同属战略布局 |
| **半导体/制造** | 半导体工艺与装备、生物制造、纤维科学与智能制造 | 微电子、材料、化工、机械 | 🟢/🟡 | 强工科院校优先 |
| **能源地矿** | 能源科学与工程、深地科学与工程、稀土科学与工程 | 能动、地质、矿业、新能源 | 🟢 | 行业院校对口度高 |
| **数字经管** | 数字金融、数字贸易、数字文旅、数据资源与数据智能 | 金融/经济+计算机、统计学、信息管理 | 🟡 | 普通院校慎追「数字」前缀 |
| **城市农业** | 城市更新、农业机器人、智慧渔业、盐碱地科学与工程 | 土木/城规、农业工程、水产养殖 | 🟡 | 看区域产业 |
| **医学交叉** | 核医学工程 | 生物医学工程、医学影像 | 🟢 | 核医学定向强 |
| **人文交叉** | 计算语言学、法律英语、心理脑与认知科学 | 计算机+中文、英语+法学、心理学 | 🟡 | 强校实验平台关键 |
| **艺术民生** | 艺术治疗、乐器智造、太极拳 | 心理学/艺术治疗研究生路径、音乐科技 | 🟡 | 就业面窄，需兴趣确认 |

**交叉学科门类**（15 种，含划入的 11 种老专业 + 4 种全新）：学位授予以目录标注为准，填报时看清是 **工学** 还是 **交叉学科** 学士学位。

## 四、推荐卡片扩展字段

2026目录新增候选在标准推荐卡片上**追加**：

```markdown
🆕 **2026目录新增** · [专业名]（首招/超常布局）
📋 **招生核验**：省计划 ✅ | 章程 ✅ | 位次数据 ❌（首招）
🏛️ **母体学科**：[院系] · [学科评估/平台]
🔀 **替代路径**：[传统专业] @ 同校或同层次
⚠️ **风险**：无往年位次；培养方案首年；就业口径未建立
```

## 五、2026 方向条目扩展（必填）

```markdown
### 2026方向条目 X：[专业名]

**目录年份**：2026 | **判定**：2026目录新增 / 首招 / 风口种草（是/否）

**权威依据**（至少 1 条）：
- 教育部目录 / 阳光高考专业库 / 省考试院计划 / 校方培养方案

**同分数段双路径实例**：
> 路径 A：…新专业… → 风险 …
> 路径 B：…传统专业… → 3 年后约 X–Y k/月（来源：就业报告/待核实）

**赛道**（见第九节）：一～八 / 待归类

**培养方案核验**（从手册提炼的检查项，须校方官网证实）：
- [ ] 核心课程是否含 …
- [ ] 是否「换名旧专业」
- [ ] 实习/实验室/行业导师

**母体学科评分**：强 / 中 / 弱 / 待核验

**ADI 摘要**（含 D 加严）：…

**志愿层级建议**：探索 / 冲 / 不得作稳保主力

**不适合如果**：数学弱、家庭不能读研、把短视频当常态、母体学科弱
```

## 六、评估模式附加项

对志愿表中每个 **2026目录新增/首招** 志愿，评估报告**额外输出**：

| 检查项 | 不合格动作 |
|--------|------------|
| 是否占稳/保主力且无位次依据 | 结构问题 → 建议下调或替换 |
| 是否仅有「专业名好听」无 ADI | 社会现实维度扣分 |
| 是否缺替代路径 | 建议补充路径 B |
| 母体学科是否弱于同校传统专业 | 建议改填传统专业 |
| 风口种草未做一天快照 | 须补 social-reality 段落 |

## 八、与家长沟通话术

| 家长/学生说 | 回应要点 |
|-------------|----------|
| 「2026目录新增是不是更有前途？」 | 前途看**产业和本校培养**，不是看名字；《目录（2026年）》新增≠本省都招 |
| 「别人家孩子报具身智能」 | 9 校超常布局，分高平台硬；普通校同名要查母体学院 |
| 「没有往年分数怎么报？」 | 只能作探索志愿，**不能占满稳保**；要有传统专业托底 |
| 「毕业能不能拿 30 万？」 | 行业顶尖岗多要硕士+一线；用中位数就业报告说话 |

## 九、八大赛道画像（推荐排序用）

> 结构借鉴市面《志愿填报指南》类手册的赛道划分，**已剔除**未经核验的薪资、缺口预测与省份分数表。推荐时按赛道选 **路径 B**，再决定是否加 **路径 A 探索**。

### 赛道一：技术硬核 · 大国重器

**涉及**：半导体工艺与装备、能源科学与工程、深地科学与工程、真空工程与技术、稀土科学与工程

| 维度 | 推荐要点 |
|------|----------|
| 定位 | 工艺/装备/能源/战略材料，偏央企国企与科研院所 |
| 母体 | 微电子、能动、地质、材料、矿业等传统强项学院 |
| 路径 B | 微电子、新能源科学与工程、地质工程、材料成型及控制 |
| 核验 | 是否真进晶圆厂/电网/地矿实习；非行业强校慎作唯一志愿 |
| 现实 | 深地/稀土多野外或资源城市；真空/半导体重物理化学 |
| 警惕 | 手册常写「起薪很高」——须查**该校**就业报告，勿用全国头部案例 |

### 赛道二：国家安全 · 公共保障

**涉及**：森林消防、政治安全保卫、核医学工程

| 维度 | 推荐要点 |
|------|----------|
| 定位 | 体制内或准体制内通道，**硬性门槛**多 |
| 路径 B | 消防/应急相关专科高职、法医学/公安技术类已有专业、生物医学工程 |
| 核验 | 体检（身高视力体能）、政审、性别与政策限制 |
| 警惕 | 仅适合已确认走体制且身体条件达标者；女性/体能不足者默认 ❌ |

### 赛道三：数字经济 · 智能经济（水分风险最高）

**涉及**：数字金融、数字贸易、数据资源与数据智能、商业人工智能、数字公共治理、数字文旅

| 维度 | 推荐要点 |
|------|----------|
| 定位 | 「专业名 + 数字化」，培养质量方差极大 |
| 路径 B | 计算机/统计/会计/信息管理 + 辅修；金融类优先有编程课的传统专业 |
| 核验 | 培养方案是否含 **Python/SQL/数据分析**；是否「旧专业换名」 |
| 商业 AI | 定位是**懂业务的 AI 应用**，不与计算机拼写码 |
| 数字文旅 | 须有真实文旅数字项目/实习，否则易空对空 |
| 数字公治 | 考公时岗位可能更偏计算机，须提前查近年职位表 |
| 警惕 | 手册称「2.3 亿数字岗位」等宏观数据**不可**用于个体志愿决策 |

### 赛道四：绿色低碳 · 资源环境

**涉及**：资源环境审计、盐碱地科学与工程、智慧渔业、智慧景观营造、交通能源融合工程

| 维度 | 推荐要点 |
|------|----------|
| 定位 | 双碳、粮食安全、智慧农业/园林 |
| 路径 B | 环境工程、审计学、农学/水产、风景园林、交通工程 |
| 核验 | 是否对接 ESG/碳核查实务；农业类看区域产业（沿海渔业、盐碱地省份） |
| 警惕 | 资源环境审计偏长期稳定，非赚快钱；智慧景观防「景观 + 皮毛 IoT」 |

### 赛道五：人文艺术 · 跨界

**涉及**：语言科学、计算语言学、语言智能、法律英语、乐器智造、太极拳、艺术治疗

| 维度 | 推荐要点 |
|------|----------|
| 定位 | 文化 + 技术/心理交叉，**强校依赖**极高 |
| 路径 B | 计算机+中文/外语、英语+法学、音乐学+电子、心理学 |
| 核验 | NLP 类看是否有语料/AI 实验室；法律英语看英语单科成绩 |
| 太极拳 | 仅建议有明确传承/就业出口者；普通院校默认 ⚠️ |
| 艺术治疗 | 市场仍在培育，短期就业不如临床心理/师范稳定 |

### 赛道六：前沿科技（超常布局集中）

**涉及**：具身智能、脑机科学与技术、生物制造、智能计算、心理/脑与认知科学

| 维度 | 推荐要点 |
|------|----------|
| 定位 | 研究型、硕士导向、硬件/实验平台要求高 |
| 路径 B | 自动化、计算机、电子信息、生物工程、心理学 |
| 核验 | 是否首批 9 校具身智能/脑机；是否有机器人或脑机实验室 |
| 具身智能 | 须接触真实硬件；代码与数学弱 → ❌ |
| 生物制造 | 合成生物深度实践，普通院校慎报 |
| 警惕 | **禁止**用招聘平台「平均年薪 33 万」类数据作本科报考理由 |

### 赛道七：交叉融合 · 产业落地

**涉及**：工程互联网、海洋智能与无人技术、农业机器人

| 维度 | 推荐要点 |
|------|----------|
| 定位 | 工业 4.0、海洋装备、智慧农业 |
| 路径 B | 自动化、物联网、海洋工程、农业机械化及其自动化 |
| 核验 | 工程互联网看工业现场实习；海洋类看是否沿海就业意愿 |
| 农业机器人 | 产业仍在放大期，**3–5 年周期**，中等分数可作长线，不宜作唯一保档 |

### 赛道八：城市更新 · 低空经济

**涉及**：城市更新、低空经济与管理、低空安全管理、纤维科学与智能制造

| 维度 | 推荐要点 |
|------|----------|
| 定位 | 存量城市更新 + 低空产业链（管理 vs 安全工学学位不同） |
| 路径 B | 城乡规划/土木/建筑学；交通运输/航空航天；纺织工程/材料 |
| 核验 | 低空经济与管理授**管理学**学位，≠ 开无人机；安全管理偏公安技术 |
| 低空 | 与 2024「低空技术与工程」区分；首招无位次 → 探索档 |
| 纤维智造 | 本质是高性能材料，非浪漫「纺织」 |

### 赛道 → 志愿层级（默认）

| 赛道 | 2026目录新增默认层级 | 稳/保主力 |
|------|----------------|-----------|
| 一、二 | 冲/探索（二须过体检政审） | 传统对口工科或已有公安消防类专业 |
| 三 | 探索为主 | 计算机/统计/会计 |
| 四、七、八 | 冲/稳（视母体） | 农学/交通/土木等传统对口 |
| 五 | 冲/探索 | 核心学科 + 辅修 |
| 六 | **探索** | 自动化/计算机/生物工程 |

## 十、商业填报手册怎么用

市面《20XX 高考志愿填报指南》类 PDF（含 **2026** 目录 38 专业解读、985/211 分数表、中外合作清单）：

| 内容类型 | 是否借鉴 | 用法 |
|----------|----------|------|
| 38 专业名单、教育部五类增设逻辑 | ✅ | 与目录 PDF 交叉核对 |
| 八大赛道、一句话定位、核心课程清单 | ✅ | 作**提问清单**，引导查培养方案 |
| 「旧瓶装新酒」「水分最大」等警惕项 | ✅ | 写入 social-reality / 家长沟通 |
| 报考建议中的**院校梯队**（微电子看西电等） | ✅ | 作母体学科核验线索，非硬编码名单 |
| 各省最低分/位次表 | ⚠️ | **禁止静态写入 skill**；用 [data-query.md](data-query.md) 查当年省考试院 |
| 薪资、缺口、市场规模预测 | ❌ | 仅作行业背景口语，**不可**作推荐依据 |
| 「黄金方向」「稳赚」「风口」话术 | ❌ | 反幻觉禁止 |

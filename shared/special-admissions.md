## 二、艺术类

### 硬门槛

1. **专业考试**：省统考合格；校考院校须校考合格（以当年简章为准）
2. **文化线**：高考文化分须达**艺术类本科文化控制线**（各省单独划线，通常低于普通批）
3. **投档规则**（各省不同，须查省招办文件）：

| 规则类型 | 含义 | 填报注意 |
|----------|------|----------|
| 文过专排 | 文化达线后按专业成绩排序 | 专业名次是核心 |
| 专过文排 | 专业达线后按文化成绩排序 | 文化位次仍重要 |
| 综合分 | 文化×权重 + 专业×权重 | 须用**当年公式**手算验证 |

### 与普通批区别

- 志愿表常在**提前批或艺术本科批**，批次与普通类**分开**
- 同一校艺术类专业录取数据**不能**与普通类专业位次混比
- 部分院校「中外合作艺术」学费极高，单独代码

### 推荐/评估要点

- P0 增加：**统考/校考分数、专业合格证、综合分**
- 无专业成绩 → **不得**用普通类位次推荐艺术院校
- ADI 的 I 维度：必须过「一天快照」（改稿、熬夜、作品集）
- 社会现实 → [arts.md](social-reality/arts.md)
- **对话实例** → [examples.md](../examples.md) **实例 8**

### 退档高发

- 文化未达艺术文化线
- 校考未过仍填报该校校考专业
- 体检受限（色盲等对设计类）

## 三、体育类

### 硬门槛

1. **术科考试**：省体育统考合格（田径、球类、体教等分项）
2. **文化线**：达**体育类本科文化控制线**
3. **投档**：多为综合分（文化+术科）或双过线后按术科/文化排序——**以省文件为准**

### 与普通批区别

- 文化线低于普通批，但**好院校竞争仍激烈**
- 体育教育、社会体育指导、运动训练等培养路径不同

### 推荐要点

- 社会现实 → [sports-education.md](social-reality/sports-education.md)
- 区分：**体育教育（师范）** vs **运动训练（竞技）** vs **社会体育（健身/教练）**
- 伤病风险、职业寿命须在方向包写明
- **对话实例** → [examples.md](../examples.md) **实例 9**

## 四、军校 / 公安 / 司法警校

### 军校（含武警工程院校等）

| 环节 | 要点 |
|------|------|
| 批次 | 通常在**提前批**，与普通本科志愿**互斥**（录取后普通批作废——以省政策为准） |
| 体检 | **军检**标准严于普通体检（视力、身高、体重、疤痕、扁平足等） |
| 政审 | 本人及家庭成员政治审查 |
| 面试 / 体能 | 部分院校有 |
| 性别 | 部分专业限男生 |

### 公安院校（公安专业）

- **公安联考**：公安院校公安学专业毕业生可参加联考当警察（政策以当年为准）
- 普通院校法学类**不等于**公安联考资格
- 体检、政审、体能测试硬性

### 司法 / 警校专科本科

- 各省警察学院、司法警官学院规则不一，**以招生章程为准**

### 推荐原则

- **未通过政审/体检预判** → 不得作为冲/稳/保推荐
- 意向必须**真实强烈**（提前批录取后往往无法参加普通批）

## 五、强基计划 / 综合评价

### 强基计划

| 项目 | 说明 |
|------|------|
| 报名 | 高考前在**阳光高考**平台报名，限少数 985 等 |
| 校测 | 高考后面试/笔试，与高考成绩加权录取 |
| 培养 | 常本硕博衔接，基础学科为主 |
| 与普通批 | 通常在**提前批或特殊批次**录取，**锁档**规则各省不同 |

### 综合评价（综评）

- 江浙沪等地常见（如昆山杜克、上纽、西交利物浦综评等）
- **高考前报名** + 校测 + 高考加权
- 学费、培养模式、毕业证与普通批不同

### 处理

- 用户仅问普通批志愿 → 提醒是否已报强基/综评，避免**重复占用**志愿或锁档冲突
- 不给「强基一定划算」结论，须结合**基础学科兴趣+长学制**

## 六、专项计划

| 类型 | 资格要点（须当年文件核验） | 志愿特点 |
|------|---------------------------|----------|
| **国家专项** | 实施区域农村户籍等 | 部分院校单列代码、有时降分 |
| **地方专项** | 本省农村户籍 | 省属高校为主 |
| **高校专项** | 农村户籍+学籍+成绩 | 阳光高考报名+公示 |
| **民族班 / 民族预科** | 少数民族身份 | 单独代码或批次 |

### 必做

- [ ] 资格是否在省招办/阳光高考**公示名单**
- [ ] 招生章程是否要求**户籍学籍满 N 年**
- [ ] 是否与定向/公费冲突

无资格 → **不得**填报专项代码（虚假填报后果严重）。

**对话实例** → [examples.md](../examples.md) **实例 13**（高校专项）

## 七、定向 / 公费

| 类型 | 履约 | 违约 |
|------|------|------|
| **公费师范** | 回生源省中小学任教约 6 年 | 退赔学费、记入诚信 |
| **定向医学生** | 基层卫生服务约 6 年 | 同上 |
| **农科定向 / 林业定向** | 基层服务期 | 同上 |
| **非西藏生源赴藏就业** 等 | 定向就业 | 合同约束 |

### 推荐原则

- 家庭**确实愿意回基层**、能接受服务期 → 可作「稳就业」方向
- 孩子强烈想去一线城市 / 考研出国 → **ADI-R 冲突**，默认不推荐
- 方向包必须写清：**服务地点、服务期、违约金额**

## 八、中外合作办学 / 境外高校

### 类型

| 类型 | 识别 | 风险 |
|------|------|------|
| 计划内中外合作 | 招生章程标注合作院校、学费 1.5–10 万/年常见 | 家庭学费、英语能力、2+2 出国费用 |
| 独立法人中外合办大学 | 如宁诺、温肯、上纽等 | 全英文、高学费、培养模式特殊 |
| 港澳高校 | 统招或独立报名 | 学费、签证、毕业证认证 |
| 「4+0」不出国 | 看章程是否真 4+0 | 部分项目仍可选 2+2 |

### 推荐原则（与 SKILL 一致）

- 家庭 **medium 以下**或明确排斥 → **默认不推荐**
- 必须标注：**年学费、是否强制出国、毕业证/学位证签发方**
- 录取位次常**低于**同校同专业普通批，勿误以为「低分捡漏同质量」

### 社会现实

- 英语弱、自律差 → 高淘汰/转学风险
- 回国就业：部分单位认，部分要求**境内 985/211 统招**

**对话实例** → [examples.md](../examples.md) **实例 12**（家庭预算与中外合作）

## 九、ADI 在特殊类型中的调整

| 类型 | ADI 调整 |
|------|----------|
| 艺术/体育 | A 看专业基础；R 看训练伤病与家庭投入；**综合分/术科为硬门槛** |
| 军警 | R 维度：体检政审一票否决 |
| 定向公费 | R 维度：服务期与地域为硬性；不合规 → ❌ |
| 中外合作 | R 维度：学费与出国费用；超预算 → ❌ |
| 专项 | R 维度：资格不满足 → ❌，不讨论 ADI 乘积 |

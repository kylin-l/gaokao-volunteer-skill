# Step 0 · 场景识别

## 模式

**A 推荐** — 从零生成方案，无志愿表。

## 决策赛道

**迷茫型** — 方向不确定；先走 guided-discovery（全景 → 逐题收窄 → 标记 → ADI），暂不按城市:专业:院校:安全权重排序。

## 特殊身份

无（普通批物理类）。

## 初步约束摘要

| 维度 | 结论 |
|------|------|
| 地域 | 一线 + 新一线（硬偏好，后续过滤） |
| 经济 | 中等 → 默认公办；中外合作/高学费须标注 |
| 排斥 | **不喜欢当医生** → 临床/口腔/长学制医学路径标 ❌；护理夜班路径谨慎 |
| 选科 | 物化生 → 理工/部分交叉可行；纯文科类受限 |
| 性别 | 女孩 — 仅在有公开供需数据时使用，不作方向截断 |

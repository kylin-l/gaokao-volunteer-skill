# 专业前景：发展 · 就业 · AI 替代性

> 考生：山西 2026 物理 622 / 位次 7204 · 主轨 CS/电子/统计 · 备轨电气  
> **说明**：2030 视角为趋势判断，非承诺；AI 替代指**本科毕业常见岗位**被工具/模型压缩的风险，不是「专业消失」。

---

## 一、读标说明

### 发展趋势（5–10 年）

| 标签 | 含义 |
|------|------|
| **增量** | 需求仍扩张，但内部分化（要能力升级） |
| **稳增** | 需求稳定偏增，偏「数据+业务」复合 |
| **绿电** | 新能源、智能电网、储能带动 |
| **国产替代** | 芯片、通信、工业软件政策红利 |
| **分化** | 头部吃饱、普通院校要靠实习/考研突围 |
| **平稳** | 省内体制/行业系统为主，涨幅有限 |

### 就业（你这个位次段，看**中位数**）

| 标签 | 含义 |
|------|------|
| **极强** | 行业校招第一梯队，实习资源好 |
| **强** | 一线/强二线好找对口岗 |
| **中强** | 要实习，名校牌更重要 |
| **中** | 属地/省会够用，全国竞争力一般 |
| **稳** | 国网/电网/国企系统，波动小 |

### AI 替代风险（本科常见去向）

| 符号 | 风险 | 典型受影响工作 | 相对安全方向 |
|------|------|----------------|--------------|
| 🟢 **低** | 难被整体替代 | — | 现场运维、电网调度、嵌入式、网安、芯片 |
| 🟡 **中** | 工具增强，初级岗压缩 | 简单页面、基础报表、重复测试 | 系统架构、业务理解、算法+领域 |
| 🟠 **较高** | 大量事务性工作可自动化 | 基础数据分析、初级会计审核 | 风控建模、合规、高端统计 |
| 🔴 **高** | 本科对口岗易被大幅压缩 | （你方案已排除营销等） | — |

## 二、你的三条主轨（详细）

### 1. 计算机 / 软件工程（志愿主力）

| 维度 | 判断 |
|------|------|
| **发展** | **增量但分化**：云、安全、工业软件、信创仍缺人；纯互联网增速放缓 |
| **就业** | 校牌+城市决定差距大：西电/北邮级 **极强**；普通 211 **中强**；省属 **中** |
| **AI 替代** | 🟡 **中**：Copilot 等压缩初级 CRUD/简单前端；**系统、后端、安全、基础软件**仍要人 |
| **5 年后图景** | 本科生更像「会用 AI 的工程协作者」，不会用 AI 的纯执行岗更难 |
| **你要注意** | 维持自学；冲④实习；别报普通校 **AI 专业** 替代 CS |
| **中位数实例** | 西电/211 CS → 西安/北京后端 3 年约 10–16k/月（城市差异大，待核实校招报告） |

### 2. 统计学 / 数据科学 / 经济统计（ADI✅）

| 维度 | 判断 |
|------|------|
| **发展** | **稳增**：金融风控、保险精算、互联网增长分析、政府数据仍要人 |
| **就业** | 央财/对外经贸 **强**；华师/晋财 **中**（地域绑定） |
| **AI 替代** | 🟡 **中**：自动报表、描述性分析受压；**因果推断、风控规则、监管合规**仍要人 |
| **5 年后图景** | 「会统计+懂业务+会编程」比纯拉数更有价值 |
| **你要注意** | 高端岗多要 **读研**；本科先占财经/数据平台 |
| **中位数实例** | 央财统计 → 银行风控/券商研究助理，北京 2–4 年约 12–18k/月（波动大） |

### 3. 电气工程及其自动化（备轨 · 国网向）

| 维度 | 判断 |
|------|------|
| **发展** | **绿电 + 智能电网**：新能源并网、储能、配电自动化长期有需求 |
| **就业** | 华电/东电/三峡等行业校 **稳**；太理电气 **山西极稳** |
| **AI 替代** | 🟢 **低**：调度辅助可 AI，**现场运维、继保、设备管理**难替代 |
| **5 年后图景** | 体制内+能源央企仍是「喜欢+稳定」最贴你 Q3 的备选项 |
| **你要注意** | 基层轮岗；部分岗位出差/抢修；女生进电网无硬性禁止但一线岗要知情 |
| **中位数实例** | 华电保定电气 → 国网山西地市 4 年含福利约 7–10k/月（父母认可型稳定） |

### 4. 电子信息 / 通信 / 集成电路（组内常作 2–3 志愿）

| 维度 | 判断 |
|------|------|
| **发展** | **国产替代**：芯片、5G/6G、车企电子、军工信息化 |
| **就业** | 西电/北邮电信 **极强**；普通校 **中强** |
| **AI 替代** | 🟢 **低**：硬件+软硬协同，偏实体与实验 |
| **和 CS 比** | 更贴你 Q1「实验算题」；项目期加班存在 |

### 5. 人工智能（仅序 8 探索 · ADI⚠️）

| 维度 | 判断 |
|------|------|
| **发展** | **分化最严重**：教培宣传>本科真实出口 |
| **就业** | 本科多落岗 **应用开发/测试**，真算法岗要硕 |
| **AI 替代** | 🟠 **较高（讽刺）**：「会调 API」的初级 AI 应用岗最先过剩 |
| **建议** | 同校 **计算机放前**；AI 仅冲 1 个探索位 |

### 6. 自动化 / 物联网（组内靠后志愿）

| 维度 | 判断 |
|------|------|
| **发展** | 智能制造、车辆电子 **稳增** |
| **就业** | **中强**，偏制造业城市 |
| **AI 替代** | 🟡 **中**：PLC/产线优化部分自动化，现场调试仍要人 |

### 7. 会计 / 审计（晋财序 42，家长备选）

| 维度 | 判断 |
|------|------|
| **发展** | **平稳**，数字化财务转型 |
| **就业** | 山西财会系统 **中**，全国竞争力弱 |
| **AI 替代** | 🟠 **较高**：凭证审核、基础报表；**审计/税务复杂业务**仍要人 |
| **和统计比** | 更稳但上限低；作家长备选合理，不作主梦想 |

## 四、和志愿策略怎么结合

| 你的诉求 | 优先看什么 |
|----------|------------|
| 喜欢 + 稳定 | 电气（🟢低替代+国网稳）与统计（稳增+体制友好） |
| 收入上限 | CS/电子 **极强** 校（序 1–2、22），接受 🟡 中替代风险 |
| 怕 AI 抢饭碗 | 少报 **人工智能** 本科；组内 **CS>AI**；电气/电信 🟢 低 |
| 愿意读研 | 统计、AI 向更划算；本科先占好学校平台 |
| 回流山西 | 太理/晋财：就业 **中** 但 **地域极强**，AI 风险看具体岗 |

# 专科批与征集志愿

> 压线生、本科滑档、艺术体育专科层时加载。普通本科批充足者仅作附录提醒。

## 一、何时进入本流程

| 信号 | 动作 |
|------|------|
| 位次贴近**专科线**或低于本科线 | 专科批为主方案 |
| 本科批**滑档** | 专科批 + 征集 |
| 用户明确「能上大专就行」 | 专科推荐 |
| 艺术/体育文化未达本科线 | 艺术/体育**专科批** |

---

## 二、专科批推荐原则

1. **防滑档优先**：保、垫占比高于本科批（保+垫建议 **50%+**）
2. **专业实用性** > 校名：护理、机电、会计、铁道、学前教育等就业导向明确
3. **专升本**通道：若孩子有升学意愿，查目标省**专升本**对口专业与比例
4. **中外合作专科**：学费高，家庭敏感默认排除
5. 仍须：**省计划 + 章程 + 体检**

### 专科冲稳保（位次参考）

| 层级 | 相对考生位次 |
|------|--------------|
| 冲 | 略高于考生（专科热门专业） |
| 稳 | 匹配 |
| 保/垫 | 明显低于考生，多所公办专科 |

## 三、征集志愿（补录）

### 是什么

每批次录取结束后，**未招满**的院校专业再次填报志愿，时间紧、计划少。

### 注意

- 分数可能**低于**常规批（不保证）
- 信息来自省招办**征集计划公告**
- 时间窗口短（常 1 天内）
- 仍须满足选科、体检

### 报告附录模板

```markdown
## 征集志愿提醒（若本科滑档）

- 关注：[省招办] 官网「征集志愿公告」
- 此前方案中可作征集候选的保底校：…
- 专科批平行志愿建议表：…
- 勿信「内部指标」诈骗
```

---

## 四、与本科方案的关系

- 本科批方案正常交付时，**附录**提醒专科批与征集，不混入本科序号
- 压线生：可**并行**准备本科冲 + 专科保

---

## 五、社会现实（高职常见专业）

| 专业 | 白话 |
|------|------|
| 护理 | 就业稳，夜班多；专升本可续本科 |
| 机电/数控 | 制造业岗位，实操强 |
| 会计 | 初级证+实操；专升本 |
| 铁道/城轨 | 行业校有优势 |
| 学前教育 | 需考证；民办园波动 |

详细就业查目标高职院校**就业质量报告**。
